# Documentation - Locator SDK - English

[< Back](../../README.md)

Welcome to the official documentation of the **DATAMOB Locator SDK**.

The Locator SDK is a **location and telemetry collection engine** for Android and iOS, with integration to **API + MQTT + WSS**, end-to-end security (**mTLS + JWT**), and **offline queue** with **exponential backoff**. It is designed for **continuous tracking** scenarios with efficient battery usage, and offers a public API (**`LocatorService`**) and an integration interface (**`LocatorIntegration`**) that decouples the SDK from your backend.

Bem-vindo à documentação oficial do **DATAMOB Locator SDK**.

O Locator SDK é um **motor de coleta de localização e telemetria** para Android e iOS, com integração a **API + MQTT + WSS**, segurança ponta a ponta (**mTLS + JWT**) e **fila offline** com **backoff exponencial**. Ele foi desenhado para cenários de **rastreamento contínuo** com uso eficiente de bateria, e oferece uma API pública (**`LocatorService`**) e uma interface de integração (**`LocatorIntegration`**) que desacopla o SDK do seu backend.
